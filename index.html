MY SKILLS

  • html

    웹 표준을 준수하는 HTML 정보 구조 설계를 통해 웹사이트 퍼블리싱을 하며, HTML5에서 권장하는 마크업 방식과 시멘틱 태그를 사용해 전체 레이아웃을 설계 할 수 있습니다.

  • CSS

    다양한 웹 환경을 고려한 크로스브라우징(Cross Browsing)을 위해 안정적인 CSS 속성을 사용하여, HTML 요소를 배치하고 디자인합니다. 또한, 기존 방식인 Float, Position, Media Query를 활용한 반응형 웹 제작 경험을 충분히 갖고 있으며, Flex와 Grid방식으로도 반응형 웹을 제작할 수 있습니다.

  • Bootstarp

    CSS 프레임웍으로 대표되는 부트스트랩에 대해 이해하고 있으며, 다양한 테마 커스텀 및 유지 보수를 통해 능숙한 사용이 가능합니다.

  • Javascript

    주로 바닐라 자바스크립트로 작업하며 필요한 경우 jQuery와 병행할 수 있습니다. 또한, 슬라이더와 같은 기능성 플러그인을 제외한 네비게이션, 탭 메뉴, 어코디언 등 상호작용을 위한 필수적인 기능은 플러그인 사용이 아닌 자바스크립트 하드 코딩으로 제작할 수 있습니다.

  • Design Tools

    Adobe Photoshop, Adobe illustrator, Figma, Zeplin, Adobe XD의 기본 기능 사용이 가능하며, 주로 Adobe Photoshop을 이용하여 간단한 편집이 가능합니다.

  • JSP

    개발자가 작성한 코드를 이해할 수 있으며, if문, for문 등 간단한 문법을 활용하여 다양한 상태에 따른 퍼블리싱이 가능합니다.
